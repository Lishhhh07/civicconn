# 🚀 Civic Issue Reporting App

A full-stack web application that allows users to report civic issues (like potholes, broken street lights, etc.), track their status, and explore nearby issues.

---

## 🌐 Live Demo

👉https://civicconn-6bet.vercel.app/

---

## ⚙️ Tech Stack

* **Frontend:** React (Vite), Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:** Vercel (Frontend), Railway (Backend)

---

## 🔥 Features

* User authentication (Login / Signup)
* Report civic issues with details
* View and explore nearby issues
* Upvote community-reported issues
* Track issue status

---

## 🛠️ Run Locally

```bash
git clone https://github.com/your-username/civicconn.git
cd civicconn
npm install
npm run dev

cd backend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the **backend** folder:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
